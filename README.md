#Phylocanvas

A canvas-based phylogenetic tree viewer, no browser plugins or extra libraries required.

Please see **www.phylocanvas.org** for documentation.

## Development
```
npm install
npm start
```
Go to [http://localhost:8000/dev](http://localhost:8080/dev)

## Building
```
npm run build
```
