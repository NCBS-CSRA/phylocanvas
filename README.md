#PhyloCanvas

A canvas-based phylogenetic tree viewer, no browser plugins or extra libraries required.

Please see **www.phylocanvas.org** for full documentation.

## Development
```
npm install
npm run dev-server
```
Go to http://localhost:8000/dev

## Building
```
npm run build
```
