#PhyloCanvas

A canvas-based phylogenetic tree viewer, no plugins and no extra libraries.

Please see **www.phylocanvas.net** for full documentation.

## Development
```
npm install
npm run dev-server
```
Test using http://localhost:8000/dev

## Building
```
npm run build
```
